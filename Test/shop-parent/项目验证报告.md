### 6.2 关键配置文件验证

**application.yml (shop-order-server):**

```yaml
server:
  port: 8091
  tomcat:
    threads:
      max: 10  # 模拟高并发场景

spring:
  cloud:
    nacos:
      discovery:
        server-addr: localhost:8848
    sentinel:
      transport:
        port: 8719
        dashboard: localhost:8858  # Sentinel控制台地址
      web-context-unify: false  # 关闭资源聚合，使链路流控生效

feign:
  sentinel:
    enabled: true  # 开启Feign对Sentinel的支持
```
